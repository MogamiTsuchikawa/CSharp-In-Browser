UMD Build Test

This page tests if the UMD build correctly exposes the CSharpBrowserCompiler class globally.

Test Code

// Test if CSharpBrowserCompiler is available globally
console.log('window.CSharpBrowserCompiler:', window.CSharpBrowserCompiler);

// Test instantiation
const compiler = new window.CSharpBrowserCompiler({
    autoInit: false, // Don't auto-initialize for testing
    onReady: () => console.log('Compiler ready!'),
    onError: (error) => console.error('Compiler error:', error)
});

console.log('Compiler instance:', compiler);
